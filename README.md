# BlackOut
Draw. Hide. Focus.

BlackOut is a Chrome extension that lets you draw, customize, and manage blackout boxes on any webpage. It is designed for quickly hiding sensitive information while taking screenshots, recording videos, or presenting content.

> 🚧 Currently under active development.

---

## ✨ Features

- Draw blackout boxes anywhere on a webpage
- Live preview while drawing
- Pin and unpin boxes
- Hide and show boxes
- Delete individual boxes
- Persistent storage across page reloads
- Per-URL persistence
- Native color picker support
- Apply a color to a single box or all boxes
- Native Chrome EyeDropper support (via the browser's color picker)

---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/Aria-vis/blackout-extension.git
```

2. Open Chrome and navigate to:

```
chrome://extensions
```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the project folder

---

## 📖 Usage

1. Click the **BlackOut** extension icon.
2. Draw a blackout box by dragging on the page.
3. Hover over a box to reveal its controls.

Available controls:

- 🎨 Change color
- 📌 Pin / Unpin
- 👁 Hide / Show
- ✕ Delete
